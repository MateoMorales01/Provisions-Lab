This is a Provisions Lab Application that helps you **navigate through menus to build and customize your sandwich, drink, and chip order**. The code provided is the entire user interface and menu structure for the application.

### Features

* **Start/Exit:** Allows the user to start a new order or exit the application.
* **Add Menu Items:** Allows the user to add a Sandwich, a Drink, or Chips to their current order.
* **Checkout/Cancel:** Provides options to complete (Checkout) or abandon (Cancel) the order.
* **Detailed Sandwich Customization:** A simple straight forward comprehensive menu that prompts for building a custom sandwich from scratch.

### Main Menu Flow

**Home Screen**
* (1) Start a new order
* (0) Exit Application

**Order Menu**
* (1) Add Sandwich
* (2) Add Drink
* (3) Add Chips
* (4) Checkout
* (0) Cancel Order

### Sandwich Customization Details

The application guides the user through detailed choices for their sandwich:
* **Bread Type:** `white, wheat, rye, wrap`
* **Size:** `4", 8", 12"`
* **Toasting:** Asks if the sandwich should be toasted.
* **Meats & Cheese:** Allows multiple selections of meats and cheeses, and prompts for **extra meat** and **extra cheese**.
* **Toppings & Sauces:** Allows multiple selections of regular toppings and sauces, and prompts for **extra sauce**.

  Drink customization:
* Pepsi only franchise

  Chips customization:
  Chips range from Cheetos, Dorritos, and Lays Products!
